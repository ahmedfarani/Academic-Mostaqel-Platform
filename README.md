## 📖 Overview

**Academic Mostaqel** is a static informational website designed to guide university students through academic programs, departments, and support services. The site presents structured content — departments, services, and study tips - through clean card-based layouts with smooth hover interactions, built entirely with HTML and CSS.

> This project was developed as a coursework assignment for the **Computer Sciences** program at **Al-Aqsa University**, under the supervision of **Dr. Musbah J. Mousa**.

## ✨ Features

- **Five content sections** | Home (hero CTA), Departments, Student Services, Academic Tips, and a Contact footer
- **Academic department grid** | four department categories (Computing & Technology, Business & Management, Engineering, Education & Humanities), each listing its specific majors
- **Student services overview** | four cards covering Academic Advising, Registration Support, Career Guidance, and E-Learning Support
- **Icon-driven tips section** | five study tip cards, each paired with a custom SVG icon (time management, active learning, goal-setting, asking for help, using resources)
- **Interactive hover states** | card lift/scale transitions on departments, services, and tips for a polished feel
- **Social contact footer** | direct email, WhatsApp, and social media links (Instagram, Facebook, LinkedIn, GitHub)

## 🛠️ Tech Stack

- **HTML5** | semantic sectioned page structure
- **CSS3** | Grid-based card layouts, custom hover transitions
- **Custom SVG icon set** | for the Academic Tips section

No build tools, frameworks, or external dependencies are required.

## 🚀 Getting Started

```bash
git clone https://github.com/ahmedfarani/Academic-Mostaqel-Platform.git
cd Academic-Mostaqel-Platform
```

Open `index.html` directly in any modern browser - no server or installation needed.
